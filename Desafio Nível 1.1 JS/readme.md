# Contador de Cliques
- Mostrar quantidade de cliques nos botões
# Público-alvo: Iniciantes

## Fluxo: 
- contador inicializado em 0,
- eventListener nos botões de + e -,
- para cada eventListener, acumular +1 ou -1 no contador,
- troca o textContent para o contador atual e remove a classe "hide" do css.

## add:
- botão de reset
- sempre que for escutado no event listener, retoma o contador a 0 e remove a classe "hide" do css.
